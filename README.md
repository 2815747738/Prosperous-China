### 项目简介

**盛世中国——展现中国公民精神风貌、宣扬中国优良传统文化、体现中国的大国责任与担当**

以中国特色社会主义事业”五位一体“总布局为主线，分别在政治、经济、文化、社会、民生五个方面中挑选一个案例展示，具体为：
政治：百年党史百年初心
经济：抗击新冠肺炎疫情
文化：中华传统节日端午节
社会：北京2022年冬奥会
生态：碳达峰与碳中和
